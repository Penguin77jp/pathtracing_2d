# 有効なレイを管理する

パストレーシングでは、8 本のレイが同時に終了するとは限りません。
各レーンが処理中かどうかを `active` マスクで管理します。

## レーンごとに異なる終了理由

- 背景へ到達した
- 材質がレイを吸収した
- 最大バウンス数へ到達した
- 他のレーンはまだ散乱を続けている

```text
lane:   0  1  2  3  4  5  6  7
active: T  F  T  F  T  T  F  T
```

## active mask の役割

- active なレーンだけヒット結果を採用する
- 終了したレーンの色や ray state を変更しない
- 散乱できなかったレーンを無効化する
- 全レーンが終了したらループを抜ける

## 代表的な処理の流れ

```text
active と hit を組み合わせる
背景へ到達したレーンを active から除く
散乱しなかったレーンを active から除く
active がすべて偽なら終了
```
