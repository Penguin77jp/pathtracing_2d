# 棄却サンプリングをパケット化する

棄却サンプリングでは、候補を作り、条件を満たさない場合に再試行します。
SIMD では、8 レーンすべてが同じ回数で成功するとは限りません。

## スカラーの場合

```text
候補を作る
条件を満たせば採用
満たさなければ再試行
```

## SIMD の場合

```text
1 回目: lane 0, 2, 5 が成功
2 回目: 未完了の lane 1, 3, 4, 6, 7 だけ再試行
3 回目: 残ったレーンだけ再試行
```

成功済みレーンを再生成すると、以前の成功値を失います。
そのため、次の状態を分けて考えます。

- 今回成功したレーン
- すでに完了しているレーン
- まだ再試行が必要なレーン

## このプロジェクトでの用途

- 単位球内部のランダム点
- 単位方向ベクトル
- カメラの単位円盤内のランダム点

## 結果の確認

範囲条件だけでは、常にゼロを返す誤った実装を見逃すことがあります。
次も確認してください。

- 複数の異なる値が出る
- 正負の両側が出る
- 平均が一方向へ強く偏っていない
- NaN や無限大がない
