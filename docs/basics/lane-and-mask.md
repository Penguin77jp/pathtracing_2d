# レーンとマスク

## レーン

このプロジェクトでは、256 ビットの SIMD レジスタを主に **8 個の 32 ビット値**として扱います。
この 8 個の独立した位置をレーンと呼びます。

```text
lane:  0   1   2   3   4   5   6   7
value: a0  a1  a2  a3  a4  a5  a6  a7
```

通常の四則演算や比較は、対応するレーン同士で行われます。
レーン 0 の計算結果が、レーン 1 に自動で移動することはありません。

このプロジェクトでは、1 レーンが 1 本のレイまたは 1 サンプルに対応します。

## マスク

比較結果は、8 個の `bool` の配列ではなく、レーンごとの全ビットを使ったマスクとして保持します。

```text
false = 0x00000000
true  = 0xffffffff
```

`Bool8` は、この 8 レーン分のマスクを `__m256` に保持しています。

```text
lane:  0  1  2  3  4  5  6  7
mask:  T  F  F  T  T  F  T  F
```

マスクは主に次の用途で使います。

- 条件に応じて値を選ぶ
- ヒットしたレーンだけ状態を切り替える
- 終了したレーンを無効化する
- 1 レーンでも処理が残っているか調べる

## マスクは数値の 0 と 1 ではない

true のマスクは `1.0f` ではありません。
32 ビットすべてが 1 のビット列です。
そのため、マスクを通常の数値として加算・乗算するのではなく、比較結果・論理演算・選択条件として扱います。

## このプロジェクトの型

| 型 | 意味 |
|---|---|
| `Float8` | 8 レーンの浮動小数点値 |
| `Bool8` | 8 レーンのマスク |
| `Vec3f8` | `Float8` を x・y・z に持つベクトル |
| `Ray8` | 8 本分のレイ |
| `HitRecord8` | 8 レーン分の交差情報 |
| `MaterialPacket8` | 8 レーン分の材質情報 |
