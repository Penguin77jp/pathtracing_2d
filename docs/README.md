# Ray Tracing SIMD Guide

このディレクトリは、intrinsic の名前を知らない状態から、
「SIMD で何をしたいか」を起点に調べるための Markdown ガイドです。

最初は [`index.md`](./index.md) を開いてください。

## 読み方

1. [`やりたいことから探す`](./index.md#やりたいことから探す) から目的を選ぶ
2. 日本語の概要とレーンの動作を確認する
3. 関連する intrinsic の名前を知る
4. リンク先の公式ドキュメントで厳密な仕様を確認する
5. [`レイトレーサの処理から探す`](./index.md#レイトレーサの処理から探す) で実コードとの関係を確認する

## 方針

このガイドは、Intel の公式仕様を置き換えるものではありません。
次の情報だけをプロジェクト側で補います。

- 何をしたいときに使うか
- SIMD の各レーンで何が起きるか
- このプロジェクトのどこで使われるか
- 初心者が間違えやすい点
- 対応する公式ドキュメント

個々の命令の正確なシグネチャ、対応 ISA、性能情報は公式資料を参照してください。
